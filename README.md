# Kart-Fury_01_ci4321
del equipo Rossaston conformado por Gastón García y por Rosa Ramirez presentamos Kart Fury. Proyecto para la materia CI4321 Computación Grafica I de la universidad Simón Bolívar. 

######### 

instrucciones de ejecucion recomendada:

1. instalar Node.js
2. instalar una herramienta de construccion: abrir una terminal en la carpeta del proyecto y colocar el comando: "npm install --save-dev vite"
3. finalizada la instalacion, colocar en la terminal: "npx vite". esto mostrará una URL en la salida.a la cual podras acceder para ver la aplicacion web

####################

instrucciones del juego:
WASD:	controles para moverse
T:	cambiar perspectiva
SPACE:	activar PowerUp
G:	cambiar a modo dios
1:	(cuando modo dios este activo) recibe powerUp "triple caparazones"
2:	(cuando modo dios este activo) recibe powerUp "Bomba"
